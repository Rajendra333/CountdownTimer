DIGITAL COUNTDOWN TIMER
A Double window digital countdown timer, built with Python Tkinter

    	***|FEATURES|***
````````````````````````````````````````````````````````````````````````````````````````
* Double Window (Control and Output)
* Works with an extended screen projection
* Maximum countdown time of 99 Minutes
* Time's Up blinking text with alarm
* Displays extra time used from the time's up period

EXTRA HACK FEATURES [UNOFFICIAL FEATURES]
* Customize a time's up text by using #YourText after label input;
	Customized text can be the only text [#YourText] in the label input box

* Customize timer fg and bg colour by using for example !@fg="#ffffff"@bg="#000000" after 
every other text in the label input box
