DIGITAL COUNTDOWN TIMER
A single screen digital countdown timer, built with Python Tkinter

    	***|FEATURES|***
````````````````````````````````````````````````````````````````````````````````````````
Maximum countdown time of 99 Minutes
Double Tap on window to toggle between fullscreen and restore window
Press Esc key to close the window
